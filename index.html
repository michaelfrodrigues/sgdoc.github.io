    O SGDoc (Sistema Gerenciador de Documentos) é um sistema para controle de documentos e processos desenvolvido em PHP e com banco de dados PostgresSQL. Foi criado como parte do modelo de gestão documental adotado pelo ICMBio (Instituto Chico Mendes de Conservação da Biodiversidade). Sendo assim este projeto se ramifica em duas partes: Contribuição com a evolução do sistema e contribuição do processo de gestão documental.

    Acesse os repositórios do projeto :: https://github.com/sgdoc
